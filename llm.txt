This is a match 3 game
I'm using airtest from github.com/AirtestProject/Airtest
the grid is the match 3 grid, when the game goes into a match, the grid show up.
It's a 8x8 grid.

this project is managed by uv. 
uv run ./KemoColiseum.py to run. 
uv add to add dependency.
uv run kemo to run.
